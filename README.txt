نظام جرد الأصول — Riyadh Schools

الملفات:
--------
server/
├── server.js       ← السيرفر الرئيسي
├── package.json    ← المكتبات
├── install.bat     ← تثبيت المكتبات (مرة واحدة)
├── start.bat       ← تشغيل السيرفر
└── public/
    ├── index.html  ← الموقع الرئيسي
    └── sign.html   ← صفحة التوقيع

خطوات التثبيت:
--------------
1. فك ضغط الملف
2. ضاعف كليك على install.bat (مرة واحدة)
3. ضاعف كليك على start.bat
4. افتح المتصفح: http://localhost:3000

إعداد الإيميل (في server.js):
------------------------------
EMAIL_USER = 'إيميلك@rsmc.edu.sa'
EMAIL_PASS = 'كلمة المرور'
EMAIL_HOST = 'smtp.office365.com'
SERVER_URL = 'http://IP-جهازك:3000'

كلمة المرور: RS@2025
